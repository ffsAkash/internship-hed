Important tips to help you hire better
1. Always assess the candidate via a short assignment or interview. Don't hire on the basis of just the application.
2. When you select a candidate, always extend a formal offer letter with key details (stipend, profile, duration etc.) & ask for a signed copy as acceptance.
3. Post that, change the selected candidate's application status to 'Hired'.
4. If you still face any problem with any applicant, kindly report it to employer@internshala.com.
